# echo-CLI
Command line interface for managing user information


### version
1.0.0

## Usage

### Installation

Install the dependencies

```sh
$ npm install
```

### Create Symlink

```sh
$ npm link
```

### Commands

List User (list or l)
```sh
$ echo-cli list
```

Find User (find or f)
```sh
$ echo-cli find [NAME]
```

Add User (add or a)
```sh
$ echo-cli add
```

Update User (update or u)
```sh
$ echo-cli update [_ID]
```

Remove User (remove or r)
```sh
$ echo-cli remove [_ID]
